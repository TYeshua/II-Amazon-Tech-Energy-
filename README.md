# 2º Amazon Tech Energy

Landing page React + TypeScript + Vite + Tailwind CSS baseada na composição fornecida.

## Executar

```bash
npm install
npm run dev
npm run build
```

Os assets ficam em `src/assets`. A foto do hero é uma alternativa gratuita do Unsplash (Charles Assunção); os logotipos foram adaptados dos arquivos fornecidos. O mapa é uma ilustração vetorial própria e substituível.
